# template-lib
Nodejs Template for libraries

This package includes the following:  
- ES-Lint
- Babel
- Mocha & chai
- Coverage Report (nyc)

## Usage
> git pull git@github.com:AkashBabu/template-lib.git
> npm install
> npm link
> yo template-lib

## Installation
> npm i lib -S
